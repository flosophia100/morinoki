# WordMap v3.2.0 機能テストガイド

## テスト対象機能

### 1. ダブルクリックによるノード作成
- **テスト方法**: 空白エリアをダブルクリック
- **期待結果**: 新しいノードが作成される
- **確認ポイント**: ノードが正しい位置に表示され、ID（node1, node2...）が付与される

### 2. Shift+クリック複数リンク生成
- **テスト方法**: 
  1. 1つ目のノードをクリックして選択
  2. Shiftキーを押しながら2つ目のノードをクリック
  3. Shiftキーを押したまま3つ目のノードもクリック
  4. Shiftキーを離す
- **期待結果**: 最初のノードから複数のノードへリンクが一括生成される
- **確認ポイント**: 
  - Shiftキーを押している間は基点ノードが保持される
  - Shiftキーを離すと操作が完了する
  - Escapeキーで操作をキャンセルできる

### 3. 自動保存機能
- **テスト方法**: 
  1. ノードを作成
  2. ブラウザを閉じる
  3. 再度開く
- **期待結果**: 作成したノードが復元される
- **確認ポイント**: 
  - データ変更時に LocalStorage に自動保存される
  - 起動時に自動保存データが復元される
  - コンソールに '[AUTO_SAVE]' ログが出力される

### 4. 無限ループの回避
- **テスト方法**: アプリを起動して何も操作せずに待機
- **期待結果**: 自動リセットのループが発生しない
- **確認ポイント**: 
  - 画面が勝手にリセットされない
  - コンソールにエラーが連続出力されない
  - CPU使用率が異常に高くならない

## エラー確認手順

### 1. コンソールログの確認
1. ブラウザの開発者ツール（F12）を開く
2. Consoleタブを確認
3. エラーメッセージや警告がないかチェック

### 2. 既存機能の動作確認
- **ノード選択**: クリックで正常に選択される
- **ノード移動**: ドラッグで位置変更ができる
- **リンク作成**: 通常のリンク作成が動作する
- **プロパティ編集**: ノード・リンクの属性変更ができる
- **保存・読込**: JSON/CSV形式での保存・読込ができる
